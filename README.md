## 🧩 Sudoku - Java Console Application

- Aplicação de Sudoku desenvolvida em Java, com foco em Programação Orientada a Objetos (POO), manipulação de coleções e boas práticas de organização de código.  
- Projeto desenvolvido como parte do Bootcamp da Digital Innovation One (DIO).  

## 🚀 Tecnologias Utilizadas  

- Java  
- Collections Framework  
- Streams API  
- Programação Orientada a Objetos  
- IntelliJ IDEA  
- Git & GitHub  

## 🎯 Objetivo do Projeto  

- O objetivo deste projeto é implementar a lógica de um jogo de Sudoku no terminal, aplicando conceitos como:  
  1. Encapsulamento  
  2. Imutabilidade  
  3. Enumeração (enum)  
  4. Separação de responsabilidades  
  5. Estruturação em camadas (model, util)  

## 🧠 Conceitos Aplicados  

- Estrutura de tabuleiro usando List<List<Space>>  
- Controle de estado do jogo com GameStatusEnum  
- Validação de erros  
- Controle de células fixas  
- Reset de partida  
- Verificação de conclusão do jogo

## ▶ Como Executar o Projeto  

1. Clone o repositório:
2. git clone https://github.com/seu-usuario/sudoku-java.git
3. Abra no IntelliJ IDEA
4. Configure o SDK para Java 17
5. Execute a classe principal

## 📌 Funcionalidades

- Inserir valor em uma posição  
- Limpar valor  
- Resetar tabuleiro  
- Verificar se há erros  
- Identificar se o jogo foi concluído  
- Exibição gráfica do tabuleiro no console
